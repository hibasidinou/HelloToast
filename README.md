# HelloToast Android Application

## Description

This project is a basic Android application developed using Java and XML in Android Studio. It was created to practice simple user interactions and understand how UI components are connected to backend logic.

The application allows the user to display a short message (Toast) and increment a counter displayed on the screen.


## Features

* A counter displayed in the center of the screen
* A button to increment the counter value
* A button to display a Toast message
* Simple user interface with custom colors and spacing


## Technologies

* Java
* XML
* Android Studio


## Project Structure

```
HelloToast/
└── app/
    └── src/
        └── main/
            ├── java/com/example/hellotoast/MainActivity.java
            └── res/layout/activity_main.xml
```


## How it works

The application contains one activity (MainActivity).

* The TextView is used to display the counter value
* When the "INCREASE" button is clicked, the counter variable is incremented and the new value is displayed
* When the "HELLO TOAST" button is clicked, a short message appears on the screen using Toast


## How to run

1. Open the project in Android Studio
2. Build the project
3. Run it on an emulator or a physical device
